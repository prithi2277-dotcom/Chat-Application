# Chat-Application
Chat Application is developed to facilitate real-time communication between users. The application demonstrates concepts such as client-server communication, message handling, user interaction, and web application development.
# Java Chat Application

## Overview
This is a simple Java Chat Application that demonstrates communication between a Server and a Client using Socket Programming.

The application allows:
- One Server
- One Client
- Sending and receiving text messages

## Requirements
- Java JDK 8 or later
- Command Prompt / Terminal
- Any Java IDE (Eclipse, NetBeans, IntelliJ IDEA) (Optional)

## Files
- ChatApp.java

## How to Compile

Open a terminal in the project folder and run:

```bash
javac ChatApp.java
